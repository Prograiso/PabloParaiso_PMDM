# PabloParaiso_PMDM

Repositorio de prácticas de PMDM (Programación Multimedia y Dispositivos Móviles) - Pablo Paraíso

## Contenido del Repositorio

Este repositorio contiene proyectos y ejercicios de desarrollo móvil con Android y Kotlin:

- **PrimerTrimestre**: Ejercicios y prácticas del primer trimestre
  - Tema 1: Fundamentos de Kotlin
  - Tema 2: Desarrollo de aplicaciones Android
- **Simulacros**: Proyectos de simulacros y práctica

## Sobre las Consultas Premium en VSCode

### ¿Por qué se agotan las consultas premium?

Las herramientas de asistencia con IA como GitHub Copilot tienen límites en el número de consultas que puedes realizar según tu plan de suscripción:

- **Plan Gratuito/Individual**: Número limitado de completaciones y consultas por mes
- **Plan Premium/Pro**: Mayor cantidad de consultas y funcionalidades adicionales
- **Plan Enterprise**: Consultas ilimitadas para organizaciones

### Alternativas cuando se agotan las consultas

1. **Esperar al reinicio del período**: Los límites se reinician mensualmente
2. **Optimizar el uso**: 
   - Usa Copilot solo para código complejo
   - Escribe código básico manualmente
   - Desactiva Copilot cuando no lo necesites
3. **Usar alternativas gratuitas**:
   - IntelliJ IDEA Community Edition (excelente para Kotlin)
   - Android Studio (IDE oficial para Android)
   - Documentación oficial de Kotlin y Android

### Configuración Recomendada para VSCode

Si estás usando GitHub Copilot en VSCode, puedes configurarlo para optimizar su uso:

```json
{
  "github.copilot.enable": {
    "*": true,
    "plaintext": false,
    "markdown": false
  },
  "github.copilot.inlineSuggest.enable": true
}
```

## Desarrollo del Proyecto

### Requisitos

- Android Studio Arctic Fox o superior
- Kotlin 1.5 o superior
- SDK de Android (API 21 o superior)

### Configuración del Entorno

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Prograiso/PabloParaiso_PMDM.git
   ```

2. Abre los proyectos en Android Studio o IntelliJ IDEA

3. Sincroniza las dependencias de Gradle

### Estructura de Proyectos

Cada proyecto contiene:
- Código fuente en Kotlin
- Recursos de Android (layouts, drawables, etc.)
- Configuración de Gradle
- Archivos de configuración de IDE (.idea)
